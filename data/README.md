# Data Directory

This directory contains datasets used for training and testing the ARAI system.

## Datasets

### SALICON
Place SALICON dataset files here for saliency prediction training.

### RICO
Place RICO dataset files here for UI component detection and classification.

### test_designs
Place your test design files here for analysis.

## Notes

- Ensure datasets are properly licensed before use
- Large datasets should be downloaded separately and not committed to git
- Add dataset files to .gitignore
